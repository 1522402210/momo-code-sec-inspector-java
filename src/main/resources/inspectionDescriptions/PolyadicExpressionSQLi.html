Momo 1001: 多项式拼接型SQL注入漏洞

攻击者可利用此漏洞，恶意修改SQL语句，导致数据库信息泄漏或被篡改。


错误实践:

sql = "select * from table where id = " + id + " and 1";


jdbc最佳实践:

sql = "select * from table where id = ? and 1";


spring-jdbc最佳实践

String sql = "select * from user where name in (:names) and status = (:status)";

MapSqlParameterSource parameters = new MapSqlParameterSource();

parameters.addValue("names", Arrays.asList("zhangsan", "lisi"));

parameters.addValue("status", 0);
